PowerShell tutorial 2: Comments

Writing comments in your program is helpful for people to understand how something works. Sometimes even you will have to go back and check something. Make sure your comments are descriptive.

Comments do not show up when you run the program, they are only visible in a source code editor.

Version 1

In version 1, only single line comments were allowed

01 | # This is a single line comment

Version 2 and up

In version 2, comment sections were introduced. These allowed for multi-line comments

01 | <# this

02 | s

03 | a

04 | multi

05 | line

06 | comment

07 | end #>
